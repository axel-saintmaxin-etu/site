L'API rest c'est quoi ? Tous simplement développé une API rest. C'est un projet en binôme qui consiste à programmer toutes les requêtes http possible d'une fausse pizzeria, c'est-à-dire tous ce qui concerne les ingrédients, les pizzas et les commandes. Cette Saé a été réalisé en tant que première Saé du 4ème semestre.

Le principe était de pouvoir utiliser les méthodes GET, POST, DELETE, PATCH et PUT sur les 3 éléments d'écrit au-dessus. On utilise Bruno pour vérifier que chaque méthodes fonctionne et renvoie les erreurs voulue.

D'un point de vue éducatif, c'est l'apprendissage de la création d'une API qui était mis en avant. La création de la documentation a également était un grand point dans ce projet. Et le travail d'équipe, la répartition des tâches sont encore les éléments principaux.
